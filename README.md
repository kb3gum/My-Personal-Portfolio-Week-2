# Kalsoom Begum- Personal Portfolio

## Description

To create my own personal portfolio based on skills gained during this bootcamp and to implement these within this project showcasing my work for future potential employers

## What I did

- Created a 4 page website linking all HTML pages
- Added a background image to main landing page to seperate this page from other linked pages
- Used semantic HTML elements on all pages
- Added nav bar section to allow user to navigate easily between pages
- Positioned nav bar in consistent location across all 3 pages including home for better accessibility
- A page containing information About Me for
- Added alt attributes to images and links
- Added Psuedo-elements to specific parts of the pages highlighting to user a defined function that allows them to move page for desired request
- Included media query on portfolio page that allows page to resize on various screens or devices
- Used variables to ensure colour consistency remained same on all pages
- Added flex box elements to page for better navigation to linked projects
- Included a contact page for any interested potential employers to complete in order to contact me
